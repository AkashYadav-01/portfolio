## Getting Started

Each template is self-contained in its own folder. Simply open the `index.html` file in a browser to preview.

### Using Tailwind CSS

These templates use Tailwind CSS via CDN for easy customization. For production use, consider installing Tailwind CSS locally:

```bash
npm install -D tailwindcss
npx tailwindcss init
```

## Customization

Each template includes:
- `index.html` - Main HTML file with Tailwind classes
- Inline JavaScript for interactions
- Responsive design for all screen sizes
